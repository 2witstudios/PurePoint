# Architecture

Architecture Decision Records (ADRs) for PurePoint's technical design.

Each page follows the ADR template:
- **Context** — Why this decision matters
- **Open Questions** — Unresolved design questions (? [DOMAIN-NNN])
- **Decisions** — Resolved questions (! [DOMAIN-NNN])
- **Design Directions** — Candidate approaches and tradeoffs
- **Research Notes** — Findings, benchmarks, prototypes
